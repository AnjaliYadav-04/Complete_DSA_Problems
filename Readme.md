## 🚀 How to Run

Make sure **Python 3** is installed on your system.

```bash
# General syntax
python folder_name/filename.py

# Examples
python patterns/pattern1.py
python recursion/basics.py
python Array/zigzag.py
```

### 🖥️ Windows — I/O Batch Runner (for recursion problems)
```bash
cd recursion
run_python_io.bat
```
